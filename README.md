This project is a simple but smart Pickleball Doubles Scoring Dashboard designed to make tracking games way easier—especially when things get confusing with serving rotations.

Instead of manually remembering who serves next, which side to stand on, or whether it’s server 1 or 2, the app handles all of that automatically. You just tap to add a point or mark a fault, and the system updates everything in real time—scores, server, positions, and side switches.

It follows actual doubles rules:

Keeps track of the two-serve system (except the first serve)
Automatically switches servers and handles side-outs
Moves players left/right only when they score
Ensures the correct serving position based on even/odd score

The dashboard is clean and visual, so you can instantly see:

Who’s serving
Which side they should be on
Current score (including server number)

Overall, it’s built for casual games, practice matches, or even small tournaments—basically removing the mental load so players can just focus on playing.
